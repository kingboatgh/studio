# **App Name**: NSP Digital Submissions

## Core Features:

- NSP Record Creation (Manual): Allow admin and desk officers to create new NSP records manually using a web form, which will automatically assign a unique ID to the created profile.
- NSP Record Creation (Bulk Upload): Allow admin and desk officers to upload multiple NSP records via Excel/CSV file.
- NSP Record Editing: Admins can edit existing NSP records, ensuring service numbers are unique.
- Monthly Submission: Mark NSP monthly evaluation forms as 'Submitted' with date, time, and officer recorded, preventing duplicates.
- NSP Search: Quick search functionality to find NSPs by ID, service number, or full name.
- Dashboard and Reporting: A simple dashboard to display totals, submitted, and pending counts. Allows exporting monthly submission reports to Excel/CSV.
- Role-Based Authentication: Simple role-based access control (Admin, Desk Officer).

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and reliability.
- Background color: Light gray (#F0F2F5) for a clean and unobtrusive backdrop.
- Accent color: A soft orange (#FFA500) for highlighting key actions and information.
- Font: 'PT Sans', a humanist sans-serif, will be used for both headlines and body text, because it's modern, readable, and suitable for a professional environment.
- Simple, clear icons to represent different actions and data categories, focusing on usability and quick recognition.
- Clean and spacious layout, optimized for desktop use with large, readable tables and minimal distractions.
- No animations to keep the experience simple and fast.
